Help

Width and height are the size of the map.

The bigger water, the more water on the map. It is not a percentage, 0.2 does not mean there will be 20% of water.

The grid type defines how to generate the cells. Voronoi is using the voronoi diagrams. Hexagons will generate a classic hexagonal grid with width * height hexagons.

Voronoi grid can be tuned. The number of sites is the init number of cells. Then, the higher relax, the rounder the cells (some might be removed).

You can specify if you want or not a coast on each side of the map. Having them all will generate an island.
